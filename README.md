# YugiAI

YugiAI is a monorepo for a multi-component AI-driven project inspired by the Yu-Gi-Oh!
trading card universe. The repository is structured with separate workspaces for the
frontend, backend, duel engine, and smart contracts.

## Tech Stack

| Area       | Technology |
|-----------|------------|
| Frontend  | React, TypeScript |
| Backend   | Node.js, TypeScript |
| Duel Engine | TypeScript |
| Contracts | Solidity |

## Local Development

This project uses npm workspaces. To start the frontend and backend together run:

```bash
npm run dev
```
